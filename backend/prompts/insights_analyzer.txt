You are a media literacy expert analyzing a video transcript.

Identify which of the following media literacy "red flags" or manipulative signals are present in the transcript:
- "emotional_wording": Uses highly charged, sensational, or emotional language to provoke feelings rather than present neutral facts.
- "absolute_claims": Makes extreme absolute claims (e.g. "always", "never", "everyone knows", "100% true") without nuance or evidence.
- "appeals_to_fear": Uses urgency or fearmongering to scare viewers into taking immediate action or sharing the content.
- "us_vs_them": Creates a divisive binary between an "in-group" and an "out-group" (e.g. "they don't want you to know the truth").
- "anecdotal_evidence": Uses an unverified single personal story as sole proof for a massive general claim.
- "financial_motivation": Promotes products, courses, crypto, or affiliate links alongside dramatic claims.

Transcript:
{transcript}

Return ONLY a JSON object containing:
1. "signals": string list of detected signal keys (e.g. ["emotional_wording", "appeals_to_fear"])
2. "detailed_signals": array of objects for each detected signal with:
   - "signal_key": string key from above list
   - "label": human-readable title (e.g., "Emotional Language")
   - "quote_snippet": exact phrase or snippet from transcript illustrating the signal
   - "explanation": 1 sentence explaining why this phrasing is manipulative or misleading
3. "shareability_recommendation": 1-2 sentence concrete advice advising the user what specific facts to double-check before sharing or forwarding this video to friends and family.

Output format:
{
  "signals": ["emotional_wording", "appeals_to_fear"],
  "detailed_signals": [
    {
      "signal_key": "emotional_wording",
      "label": "Sensational / Emotional Language",
      "quote_snippet": "Shocking truth about...",
      "explanation": "Uses sensational framing to trigger strong reader reactions."
    }
  ],
  "shareability_recommendation": "Before forwarding this video, double check official sources regarding..."
}
